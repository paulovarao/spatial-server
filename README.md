## Services

### 1) Points

* Rotation around a centroid
* Linear interpolation
* Circular buffer
* Rectangular buffer
* Distance calculation (considering the Earth as a spheroid)

### 2) Lines

* Buffer

### 2) Polygons

* Area
* Intersection between polygons
* Mosaic
* Simplification of non simple polygons
* Union of polygons
